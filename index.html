How to Use Co Pilot Collections Manager

Simple daily workflow Keep collectors focused: search, call, status, note, promise, follow-up, next account.
1. Import accounts

Admin imports a CSV, maps columns, checks the account list, then assigns accounts to employees.

2. Work the queue

Collectors open their queue, search primarily by phone number, click a debtor, call, status, and notate.

3. Use Today

The Today Dashboard shows callbacks, promises due, overdue payments, and broken promises.

4. Payment plans

Create payment plans tied to the current balance. Record payments so balances and plan totals update.

5. Generate letters

Create debtor letters/PDFs with the company logo watermark. Use fake data for demos.

6. Manage staff

Admin approves employees, assigns accounts, monitors activity, and removes fired employees.
